# SUCCESS METRICS

## Product & Business KPIs
- User Adoption (DAU/MAU): Target 10k MAU in 12 months.
- Trial → Paid Conversion: Target 8–12% for team plans.
- Retention: 6-month retention for paid users >= 70%.

## Usage & Performance Metrics
- Meetings Processed per User per Month: Track baseline and adoption impact.
- Average Meeting Processing Time: Target <15s (AI) / <10s (fallback) for <=5k words.
- AI Response Time: Median external API <3s; end-to-end <=15s.
- Fallback Success Rate: >=95% when Anthropic unavailable.

## System & Reliability Metrics
- System Availability: >=99.9% for core endpoints.
- Search Performance: 95th percentile <=2s.
- Export Success Rate: 99%.

## User Satisfaction
- NPS: Target >=40 for paying customers.
- CSAT: >=85% for summary accuracy surveys.

## Business KPIs
- ARR / MRR and ARPU tracked monthly.
- Churn: Monthly churn <3% for paid customers.

## Technical KPIs
- AI Cost per Summary: Monitor and optimize.
- Queue Depth: Alert thresholds for worker backlogs.
- Error Rate per 1,000 requests: Monitor and reduce.

## Measurement
- Dashboards in analytics tools (Grafana/Looker) with weekly and monthly reviews.
